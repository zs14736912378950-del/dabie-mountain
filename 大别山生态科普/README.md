# 🌲 大别山生态密码（修复版）

互动科普H5 — 探索安徽大别山森林生态系统

## 🎯 项目简介

**大别山生态密码**是一款面向青少年的互动科普H5应用，通过游戏化的交互体验，让用户在探索中学习大别山的生态知识。项目采用纯前端技术实现，无需后端服务，支持离线访问。

### 核心特色

- **🎬 电影级开场**：沉浸式首页设计，粒子动画与视觉特效
- **🌲 4层视差森林**：SVG绘制的多层视差滚动场景
- **🐾 12个物种热点**：交互式物种发现系统，配有趣味知识卡片
- **🧩 食物链拼图**：拖拽式营养级排序游戏
- **⚠️ 危机抉择**：情境分支选择，影响生态值评分
- **📝 知识测验**：6道生态知识问答题
- **🏆 守护者证书**：基于表现的等级评定与证书生成
- **🎙️ 语音导读**：Web Speech API 实现页面内容朗读
- **🔊 音效系统**：Web Audio API 合成环境音效
- **📱 PWA支持**：Service Worker 离线缓存

## 🏗️ 技术架构

```
技术栈：Vanilla JS + GSAP + Web Audio API + Web Speech API
构建工具：Vite
设计系统：CSS Custom Properties（设计令牌）
动画引擎：GSAP (GreenSock Animation Platform)
```

## 🎮 功能模块

### 1. 首页 — 电影级开场
- 黑色遮罩淡出动画
- 背景图预加载与骨架屏
- 飘落粒子效果（🍃✨🌿💚⭐）
- 标题彩蛋：连点3次触发星星雨
- 引导角色"小林"入场对话

### 2. 大别山地理知识
- 英雄区全景展示
- 核心数据卡片（3000+植物、400+鸟类等）
- Canvas生物多样性柱状图
- 四季变化卡片
- 自然保护区列表
- 保护历程时间线
- 真实保护故事

### 3. 森林探索 — 4层视差
- **天空层**：云朵漂移、太阳光线
- **远山层**：三层山脉叠加
- **森林层**：SVG树木矩阵
- **前景层**：灌木、溪流、蘑菇、落叶
- 12个交互热点（物种发现系统）
- 飞鸟动画、溪流波光
- 落叶飘落效果
- 物种知识卡片（Bottom Sheet）
- 彩蛋：连续点击大树5次出现松鼠🐿️

### 4. 食物链拼图
- 拖拽排序（支持桌面端和移动端）
- 7个营养级槽位
- 干扰项设计（增加难度）
- 能量流动动画（验证成功后）
- 触觉反馈支持

### 5. 危机抉择
- 8个生态危机情境
- 分支选择影响生态值
- SVG场景插图
- 连续正确奖励（金色粒子）
- 错误选择灰度效果

### 6. 知识测验
- 6道选择题
- 即时反馈与解析
- 成绩评定系统

### 7. 守护者证书
- 4个等级：青铜→白银→黄金→钻石
- 雷达图素养评估
- Canvas证书图片生成
- 分享海报生成
- 真实保护故事展示
- 行动号召引导

## 🎨 设计系统

### 色彩体系
```css
--color-primary: #059669     /* 自然绿 */
--color-accent: #FBBF24      /* 暖金色 */
--color-success: #059669     /* 成功 */
--color-danger: #DC2626      /* 危险 */
--color-warning: #D97706     /* 警告 */
```

### 设计原则
- **有机圆润**：大圆角设计（16-24px）
- **Claymorphism风格**：粘土质感按钮
- **毛玻璃效果**：导航栏与进度条
- **物理弹性**：自然缓动曲线
- **8dp节奏**：统一间距系统

### 动画规范
```css
--ease-out: cubic-bezier(0.16, 1, 0.3, 1)      /* 减速 */
--ease-spring: cubic-bezier(0.34, 1.56, 0.64, 1) /* 弹性 */
--duration-fast: 150ms    /* 快速反馈 */
--duration-normal: 250ms  /* 标准过渡 */
--duration-slow: 400ms    /* 慢速动画 */
```
